# Sample Node Application for the EC2 Micro Series

This is the sample Node.js, Loopback used in the AWS DevOps EC2 Micro Series.  To use:

1. Make a directory for it

```
mkdir app && cd $_
```

2. Clone it

```
git clone <url> .
```

3. Install the dependencies

```
npm install
```

4. Run it:

```
node .
```
